# Apresentando o Back-End

<br />

### Introdução

<p>
  A escolha da linguagem que utilizamos no Back-End foi NodeJS por questão de experiência.
</p>

### NodeJS

<p>
  O <strong>NodeJS</strong> é uma plataforma de aplicação <strong>Server-Side <i>(Back-End)</i></strong> onde você
  utiliza a linguagem de programação JavaScript onde esse códigos serão compilados, otimizados e interpretado por uma
  máquina virtual.
</p>

### ExpressJS

<p>
  O ExpressJS é um framework para aplicações web para Node.js, lançado como software livre e de código aberto sob a
  Licença MIT. É
  feito para otimizar a construção de aplicações web e API's. É um dos mais populares frameworks para servidores em
  Node.js.
</p>

### Middleware CORS

<p>
  CORS - Cross-Origin Resource Sharing (Compartilhamento de recursos com origens diferentes) é um mecanismo que usa
  cabeçalhos adicionais HTTP para informar a um navegador que permita que um aplicativo Web seja executado em uma origem
  (domínio) com permissão para acessar recursos selecionados de um servidor em uma origem distinta. Um aplicativo Web
  executa uma requisição cross-origin HTTP ao solicitar um recurso que tenha uma origem diferente (domínio, protocolo e
  porta) da sua própria origem.
</p>

### Simulando a comunicação da API com Banco de Dados

<p>
  Geralmente para projetos onde o foco não é a comunicação com o Banco de Dados, escolhi utilizar um Objeto JSON
  <i> (JavaScript Object Notation)</i>.
</p>

<ul>
  <li>
    <p>Leitura mais simples</p>
  </li>
  <li>
    <p>Analisador(parsing) mais fácil</p>
  </li>

    <p>JSON suporta objetos!</p>
  </li>
